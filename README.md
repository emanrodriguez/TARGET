# TARGET

## Requirements
1. Must download latest version of ChromeDriver https://chromedriver.chromium.org/downloads (Make sure to copy file path)
1. Package requirements (**OPTIONAL, RUN THE *"pipInstall.py"* FILE TO INSTALL ALL PACKAGES**):
   1. tqdm
   1. selenium
   1. tkinter
   1. json
